# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-09

### Added
- Dark fantasy narrative game with AI-powered Game Master
- 5-chapter story arc with the Crown of Twilight quest
- 4 possible endings (Heroic, Corrupted, Sacrifice, Tragic)
- Groq and Google Gemini API support
- Procedural ambient audio engine (drone, wind, sparse piano, reverb)
- Choice sound effects and death sound
- Mute/unmute toggle
- Save/resume system with localStorage
- Quit confirmation modal (save & quit, restart, continue)
- Back button interception to prevent accidental page leave
- Stat system (Health, Sanity, Gold) with danger animations
- Drop cap first letter styling for narrative text
- Animated floating golden particles on title screen
- Multi-layer atmospheric background (fog, gradients, noise texture)
- Full French and English translation (i18n)
- Language selector on title screen
- Responsive design (mobile, tablet, desktop, landscape)
- Touch device optimizations (larger tap targets, no hover effects)
- Safe area support for notched phones (iPhone X+)
- Victory screen with ending title, epilogue, and final stats recap
